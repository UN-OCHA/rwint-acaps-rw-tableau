ReliefWeb Tableau WDC

DISASTERS WDC
This Web Data Connector requests a set of Disasters that ocurred by country and between dates defined by the user.
Data is extracted from the ReliefWeb API.
JOBS WDC
This Web Data Connector requests a set of Jobs that occurred by country and between dates defined by the user.
Data is extracted from the ReliefWeb API.
REPORTS WDC
This Web Data Connector requests a set of Reports that ocurred by country and between dates defined by the user. Data is extracted from the ReliefWeb API. Both primary and secondary country of the report are included in the download, which increases the number of requests and the time required to download it. If you are interested only by the primary country, use the Report_NC WDC instead.
REPORTS_NC WDC
This Web Data Connector requests a set of Reports that ocurred by country and between dates defined by the user.
Data is extracted from the ReliefWeb API.
